# AnimeStream - Free Anime Streaming App

Watch your favorite anime episodes directly in your browser!

## Features
-  Watch anime episodes
-  Search functionality
-  Mobile responsive
-  Easy to use interface

## Admin Panel
- Password: `deepak@#509`
- Add new anime with YouTube video IDs
- Delete existing anime

## How to Use
1. Open the website
2. Click on any anime card
3. Video starts playing automatically

## Technology Used
- HTML/CSS/JavaScript
- Tailwind CSS
- Firebase Realtime Database
- YouTube IFrame API
